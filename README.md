# only-stackoverflow

- 검색 결과에서 스택오버플로 사이트 긁어다가 번역기 돌려놓은 이상한 사이트들을 모조리 숨겨요

- [구글](https://google.co.kr)과 [덕덕고](https://duckduckgo.com/)를 지원해요.

- 이슈/풀리퀘 매우 환영

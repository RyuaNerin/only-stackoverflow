## only-stackoverflow 필터 구독하기

- **테스트된 플러그인**
    - **AdGuard**
    - **uBlock Origin**
    - **Personal Blocklist(not by Google)**
    - **Adblock Plus**
    - **AdBlock**

- Adblock Plus, AdBlock 사용시 주의사항
    - (Adblock Plus) "허용되는 광고"를 허용하거나 (AdBlock) "허용할만한 광고" 필터를 구독하면 google.com에서 모든 필터가 적용되지 않습니다. 이를 해제해야 합니다.
